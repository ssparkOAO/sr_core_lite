SR Core RTL_sys Module Relationship Map
======================================

Purpose
-------
This file documents how the current Vivado streaming project, top modules,
testbenches, RTL_sys modules, verified RTL modules, and Vivado BMG IP blocks are
connected.

Read this before opening Vivado if you want to know where each source file lives
and which testbench verifies which top.

5. Vivado Source Snapshot Rule
------------------------------

The streaming Vivado project should use:

  vivado/sr_core_streaming_pynqz2/src/rtl/*.v
  vivado/sr_core_streaming_pynqz2/src/tb/tb_sr_nn_top_img.v

The snapshot is not the golden source of truth.

If a verified RTL module or RTL_sys architecture module changes later:

  1. Update and verify the original source in RTL/ or RTL_sys/.
  2. Refresh the copied file under vivado/sr_core_streaming_pynqz2/src/.
  3. Update vivado/sr_core_streaming_pynqz2/src/manifest/SOURCE_SNAPSHOT_MAP.txt.
  4. Re-run Phase9.5 image-level verification.
