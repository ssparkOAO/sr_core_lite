Phase9.2 新增兩個 RAM IP prototype： conv3_feature_ram 和 output_image_ram。

步驟3：conv3_feature_ram

GUI OptionSelected ValueWhyEquivalent Tcl
Component Nameconv3_feature_ram存 Conv3 output tensor，不再叫 feature_mem1-module_name conv3_feature_ram
Memory TypeSimple Dual Port RAMPort A 寫 Conv3 output，Port B 讀給 PixelShuffleCONFIG.Memory_Type {Simple_Dual_Port_RAM}
Write Width A32一個 LR pixel 存 4 個 int8 channel：{c3,c2,c1,c0}CONFIG.Write_Width_A {32}
Read Width A32一次讀回同一 pixel 的 4 channelsCONFIG.Read_Width_A {32}
Write Depth A64目前 LR tensor 是 8x8，共 64 pixelsCONFIG.Write_Depth_A {64}
Enable A/BUse EN Pin讓 controller 明確控制 RAM accessCONFIG.Enable_A/B {Use_EN*_Pin}
Output RegisterDisabled先保持和 behavioral RAM 一拍讀取模型接近Register_Port*_Output... {false}

步驟4：output_image_ram

GUI OptionSelected ValueWhyEquivalent Tcl
Component Nameoutput_image_ram存 final uint8 image output，不再叫 feature_mem2-module_name output_image_ram
Memory TypeTrue Dual Port RAMPixelShuffle 同一 cycle 可能寫 even row 和 odd rowCONFIG.Memory_Type {True_Dual_Port_RAM}
Write Width A/B16一個 word 存兩個水平相鄰 uint8 pixelsCONFIG.Write_Width_A/B {16}
Read Width A/B16日後讀出 image 時同樣以 packed word 讀CONFIG.Read_Width_A/B {16}
Write Depth A12816x16 output image，每 word 2 pixels，所以 256/2 = 128CONFIG.Write_Depth_A {128}
Enable A/BUse EN Pin清楚表示 write transaction 有效時間CONFIG.Enable_A/B {Use_EN*_Pin}
Output RegisterDisabled先降低額外 latency，方便和 behavioral model 對齊Register_Port*_Output... {false}

步驟5：Tcl 對照

create_ip -name blk_mem_gen -vendor xilinx.com -library ip -version 8.4 -module_name conv3_feature_ram
set_property -dict [list CONFIG.Memory_Type {Simple_Dual_Port_RAM} ...] [get_ips conv3_feature_ram]

create_ip -name blk_mem_gen -vendor xilinx.com -library ip -version 8.4 -module_name output_image_ram
set_property -dict [list CONFIG.Memory_Type {True_Dual_Port_RAM} ...] [get_ips output_image_ram]

注意事項

Phase9.2 的 RTL wrapper 先 expose raw RAM ports，testbench 使用 behavioral RAM model。 Vivado RAM IP 已在新 project 中建立，下一步可以把 behavioral RAM model 換成 Vivado BMG simulation wrapper， 或進一步建立 block design 讓 CNN core 和 RAM IP 同層接線。
