4. Phase9 Project Split

Phase9 的重點是分清楚 behavioral verification 與 Vivado IP-backed verification。 Behavioral model 用來快速 debug controller；Vivado project 用來確認 BMG ROM/RAM IP wrapper 與 schematic hierarchy。

路徑 用途
RTL_sys/phase9_3_ref_hier/ Reference-like hierarchy behavioral verification。
RTL_sys/phase9_4_ip_verify/ BMG IP-backed verification。
RTL_sys/phase9_5_image_test/ 128x128 image-level SR inference test。
vivado/sr_core_streaming_pynqz2/ Streaming architecture Vivado project，包含真正 BMG ROM/RAM IP。
