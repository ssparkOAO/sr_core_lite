1. 新 IP 與功能

IP 功能
ZYNQ7 Processing System 提供 PS、DDR、FCLK、M_AXI_GP0，以及 DMA 存取 DDR 用的 S_AXI_HP0。
AXI DMA MM2S 把 input 從 PS DDR 送到 PL；S2MM 把 output 從 PL 收回 PS DDR。
AXI Interconnect 或 SmartConnect 把 PS AXI-Lite 控制交易分到 DMA register 和 wrapper register。
Processor System Reset 把 PS reset 轉成 PL 端可用的同步 reset。
sr_core_ps_dma_wrapper Phase10.1 到 10.5 完成後加入的自訂 wrapper IP。

2. GUI 建立順序

  1. 建立新專案 sr_core_phase10_ps_dma_pynqz2，不要開舊的 clean-stream project 改。
  2. 建立 block design：sr_core_phase10_bd。
  3. 加入 ZYNQ7 Processing System，Run Block Automation，使用 PYNQ-Z2 preset。
  4. 確認 FCLK_CLK0 啟用，v1 先用 100 MHz。
  5. 確認 M_AXI_GP0 和 S_AXI_HP0 啟用。
  6. 加入 AXI DMA，關閉 Scatter Gather，保留 MM2S 和 S2MM。
  7. 加入 AXI Interconnect 或 SmartConnect。
  8. 加入 Processor System Reset。
  9. 加入 Phase10 wrapper IP：sr_core_ps_dma_wrapper。
  10. 完成拉線與 Address Editor 後，最後才 validate design。

8. PYNQ v1 操作方向

overlay = Overlay("sr_core_phase10_ps_dma_pynqz2.bit")
dma = overlay.axi_dma_0
ctrl = overlay.sr_core_ps_dma_wrapper_0

in_buf = allocate(shape=(128 * 128,), dtype=np.uint8)
out_buf = allocate(shape=(256 * 256,), dtype=np.uint8)

dma.recvchannel.transfer(out_buf)
dma.sendchannel.transfer(in_buf)
ctrl.write(0x00, 1)
dma.sendchannel.wait()
dma.recvchannel.wait()
done = ctrl.read(0x04)

v1 使用 polling，不先接 interrupt。
