SR Core Streaming Vivado Source Snapshot Map
===========================================

Purpose
-------
This folder is a Vivado project source snapshot.

It is used so the streaming Vivado project can show all Phase9.5 source files
under one local src/ tree instead of requiring the user to jump between RTL/,
RTL_sys/phase9_3_ref_hier/, and RTL_sys/phase9_5_image_test/.

Important rule:

  RTL/ remains the Golden Verified RTL Module Library.
  RTL_sys/ remains the architecture development workspace.
  vivado/sr_core_streaming_pynqz2/src/ is only a project snapshot.

If the original RTL or RTL_sys source changes later, refresh this snapshot and
update this file.


Current Vivado Project
----------------------
Project:
  vivado/sr_core_streaming_pynqz2/sr_core_streaming_pynqz2.xpr

Design top:
