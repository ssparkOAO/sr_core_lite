SR Core Clean Stream Vivado Source Snapshot Map
==============================================

Purpose
-------
This folder is a Vivado project source snapshot for:

  vivado/sr_core_clean_stream_pynqz2

It is not the golden source library.

Original source ownership:

  RTL/      -> Golden Verified RTL Module Library
  RTL_sys/  -> Architecture development workspace
